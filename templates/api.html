API Documentation

Integrate file conversion into your apps with our simple REST API

Convert File

POST /convert
ParameterTypeDescription
fileFileThe file to convert
target_formatStringTarget format (e.g., jpg, png, pdf, mp4)
qualityIntegerQuality 1–100 (default 85)

Remove Background

POST /remove-bg
ParameterTypeDescription
fileFileThe image file
formatStringOutput format: png, jpg, webp

Compress File

POST /compress
ParameterTypeDescription
fileFileThe file to compress
qualityIntegerCompression quality 1–100

Batch Convert

POST /convert/batch
ParameterTypeDescription
filesFile[]Multiple files
target_formatStringTarget format for all files

Example

curl -X POST https://image-converter-lake.vercel.app/convert \
  -F "file=@document.pdf" \
  -F "target_format=jpg" \
  -F "quality=90"
