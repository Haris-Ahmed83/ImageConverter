API Documentation

Convert images programmatically with our simple REST API

Convert Image

POST /convert
ParameterTypeDescription
fileFileThe image file to convert
target_formatStringTarget format: jpg, png, webp, gif, bmp, tiff, ico, avif
qualityIntegerQuality (1–100, default 85)

Remove Background

POST /remove-bg
ParameterTypeDescription
fileFileThe image file
formatStringOutput format: png (default), jpg, webp

Compress Image

POST /compress
ParameterTypeDescription
fileFileThe image file to compress
qualityIntegerCompression quality (1–100)

Batch Convert

POST /convert/batch
ParameterTypeDescription
filesFile[]Multiple image files
target_formatStringTarget format for all files

cURL Example

curl -X POST https://image-converter-lake.vercel.app/convert \
  -F "file=@image.heic" \
  -F "target_format=jpg" \
  -F "quality=90"
